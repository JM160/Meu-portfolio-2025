# Portfólio pessoal - João Matheus
Este repositório contém meu portfólio pessoal online, desenvolvido para apresentar minhas habilidades, estudos e projetos em desenvolvimento web front-end, com foco em HTML5 e CSS3 puros.

O site foi projetado para ser responsivo, acessível, visual­mente limpo e bem estruturado, servindo tanto como projeto acadêmico quanto como vitrine profissional.

# Contexto do Projeto

Este projeto foi desenvolvido como parte do Desafio Relâmpago 01 da trilha de UI/UX do programa Alpha Ed-Tech.

O desafio teve como objetivo aplicar, na prática, os conceitos estudados ao longo da trilha, incluindo:

- Prototipagem de interfaces

- Estruturação semântica em HTML

- Design visual limpo e consistente

- Responsividade para múltiplas resoluções

- Acessibilidade web

- Boas práticas de organização de código

Todo o desenvolvimento foi realizado utilizando apenas HTML5 e CSS3, sem o uso de frameworks ou bibliotecas externas, conforme os requisitos do desafio.

# Tecnologias usadas

HTML5:
- Estruturação semântica do conteúdo
  
- Uso de tags como header, nav, main, section, article, footer

CSS3:
- Layouts com Flexbox e Grid

- Responsividade para diferentes resoluções
  
- Variáveis CSS para padronização visual

Figma:
- Criação de um protótipo visual do site antes da implementação

# Design e Prototipação
O layout do site foi previamente planejado e prototipado no Figma, respeitando princípios de:

- Hierarquia visual

- Contraste e legibilidade

- Espaçamento consistente

- Simplicidade e clareza de navegação

O design final segue fielmente o protótipo, priorizando uma estética moderna, minimalista e funcional.

A prototipação do Figma pode ser acessada por esse link:
https://www.figma.com/design/fBrbvQSBPxLRlv4bFCaRpu/Meu-portf%C3%B3lio?node-id=0-1&t=ArJHtWtvYDIPxTe4-1

# Responsividade
O site foi desenvolvido para se adaptar corretamente às seguintes resoluções:

- 4K (Ultra HD)

- 2K

- Full HD

- HD

- SD / dispositivos móveis

A responsividade foi implementada utilizando media queries, unidades relativas (rem, %, vw, vh) e layouts flexíveis.

# Acessibilidade
- Foram adotadas boas práticas de acessibilidade, incluindo:

- Uso correto de tags semânticas

- Textos alternativos em imagens (alt)

- Navegação clara e previsível

- Contraste adequado entre texto e fundo

- Estrutura compatível com leitores de tela
